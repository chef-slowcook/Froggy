# Frogg Swing
Game about swinging parkour frogs 🐸
## Controls
### Jump
- **Right Click + Hold** = Charge Jump

- **Right Click + Release** = Jump

### Tongue
- **Left Click** = Project Tongue

- **Left Click + Release** = Retract Tongue
